Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • JavaScript je vysokoúrovňový interpretovaný skriptovací jazyk.
  • Používá se pro interaktivitu, dynamické funkce na webových stránkách.
  • Syntaxe JavaScriptu je podobná jiným programovacím jazykům, jako je C, C++, a Java, což usnadňuje učení a používání.
  • JavaScript umožňuje manipulaci s webovými stránkami a dokumenty, včetně změny obsahu, přidávání událostí a reakcí na uživatelské interakce.
  • Mimo webový prohlížeč je JavaScript také používán na serverové straně pomocí platformy Node.js, což umožňuje psát serverový kód v JavaScriptu.
  • Existuje mnoho knihoven a frameworků, které usnadňují vývoj v JavaScriptu, jako například React, Angular, Vue.js pro vývoj uživatelského rozhraní a Express.js pro tvorbu serverového kódu.
  • JavaScript je podporován většinou moderních prohlížečů a není omezen na konkrétní platformu, což umožňuje psát kód, který běží na různých zařízeních a prohlížečích.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Consistent Indentation: Use a consistent and readable indentation style, typically with 2 or 4 spaces per level.
  2. Meaningful Variable and Function Names: Choose descriptive and meaningful names for variables and functions to improve code readability and understanding.
  3. Use Comments: Include comments in your code to explain complex logic, functions, or any parts that may not be immediately obvious.
  4. Follow a Style Guide: Adhere to a recognized JavaScript style guide, such as Airbnb, Google, or the Standard Style, to maintain a consistent coding style across your project.
  5. Error Handling: Implement proper error handling to gracefully handle unexpected issues and prevent crashes, and log meaningful error messages for debugging.

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // Celé číslo (typ number) s přiřazenou hodnotou
let y; // Bez přiřazení hodnoty, vytvoří se jako undefined
let z = null; // Null, což je zvláštní hodnota, která označuje "prázdnost" nebo "neexistence hodnoty"
let number = 42.5; // Reálné číslo (typ number) s desetinnou částí
let exponentialNumber = 1.23e-4; // Zápis desetinného čísla pomocí exponentu
let text = "Hello, world!"; // Řetězec (typ string)
let isLeapYear = false; // Boolovská hodnota (typ boolean) pro logický výraz (true/false)

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, /=, *=, %=, ++, --, += (pro spojování řetězců)

Logické

&& (a zároveň), || (nebo), ! (negace), != (nerovná se), ? : (ternární operátor)

Porovnávací

== (rovná se), === (strikně se rovná, posuzuje i datový typ) Pro porovnávání čísel: < (menší), <= (menší nebo rovno), > (větší), >= (větší nebo rovno) Větší a menší se také používá k porovnávání lexikografického pořadí

Bitové

&, |, ~, ^, <<, >>, >>>

Typové

typeof (typ proměnné), instanceof (), cosntructor

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Zobrazí jednoduché informační okno s textovou zprávou. Používá se k informování uživatele o nějakém důležitém faktu nebo události. */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
Zobrazí okno, kde uživatel může zadat textový vstup. Výsledek je často uložen do proměné password.

confirm()

/* Zobrazí se popup okno, které se uživatele ptá, jestli chce přesměrovat na stránku seznam.cz */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:

  1. Omezení způsobená blokováním: Moderní prohlížeče mohou blokovat popup okna, což způsobuje, že uživatelé je nemusí vidět.
  2. Rušení uživatelů: Příliš mnoho popup oken může rušit uživatele a vést k negativnímu vnímání stránky.
